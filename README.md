# Cohab-chat
Сервис чата для основного сервиса Cohab на реактивном стеке webflux с использованием MongoDB как хранилища и Redis как брокер сообщений. 
Вероятнее всего данный сервис будет в основе использовать RSocket протокол, который поддерживают современные браузеры.
## Желаемый функционал:
- Пользователи умеют отправлять сообщения в семейный чат
- Пользователи умеют отвечать на чужие сообщения
- Пользователи умеют отправлять видео в семейный чат
- Пользователи умеют просматривать кто прочитал их сообщение
- Пользователи умеют ставить реакции на сообщения
- Уведомления о сообщениях прямо в браузер
- Центр активности с отслеживанием когда человек был в сети и в каком сейчас состоянии(онлайн, оффлайн, отошел - определяет система)
